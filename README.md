# Sistema de Gestión de Biblioteca (Desafío #1)
Aplicación de escritorio realizada con Winforms y lenguaje C# para gestionar libros, usuarios y préstamos.

Tecnologías utilizadas:
- C#
- .NET Framework
- Git
- Windows Forms

##Funcionalidades 
- CRUD de Libros
- CRUD de Usuarios
- Registro de préstamos
- Estadisticas con gráficas
